# automatizacion-de-registro-de-ventas

realice una funcion para todo, en donde un while con una variable tipo booleana, para lograr ingresar los productos que quisiera el usuario todas las veces que quisiera.
luego como antes inicialice un diccionario llamado sales, en donde voy a guardar toda las ventas
despues pedi por teclado, el nombre el precio y la cantidad
luego con un contador que hice, logre poner como key cada numero del contador, iniciando desde uno
ya cuando el usuario no quiere ingresar mas ventas, se le muestra tal y como lo exige la historia de usuario todos los producos, sus cantidades y la suma total de los productos vendidos.
